# LangChain整体介绍

- 基于大语言模型构建端到端语言模型应用的框架，提供工具、套件和接口，支持文本生成、文档问答、聊天机器人等复杂任务。

* 中文文档: https://langchain-doc.cn/v1/python/langchain/overview.html
* GitHub源码: https://github.com/langchain-ai/langchain
* 原版文档：https://docs.langchain.com/oss/python/langchain/overview

核心功能：

1. 模型（Models）

   功能：包含各大语言模型的接口和输出解析机制

   版本变化：1.0版本对消息格式进行了重大更新，支持多模态输入
2. 提示模板（Prompts）

   功能：让提示词工程流线化，进一步激发大模型的潜力
3. 记忆（Memory）

   通过短时记忆和长时记忆，在对话过程中存储和检索数据，让ChatBot记住你
4. 数据检索（Indexes）

   **功能**: 构建操作文档的方法，实现本地知识库搭建

   **相关模块**:

   * Retriever(向量检索)
   * Document Loader(文件加载)
   * Embedding Model(文本向量化)
5. 链（Chaains）

   **现状**: 1.0版本中已标记为弃用，可通过  `langchain-classic` 包继续使用

   **历史作用**: 曾是核心机制，用于封装特定功能并灵活组合任务
6. 代理（Agnets）

   **核心地位**: 1.0版本的主要抽象机制

   **功能特点**:

   * 自主调用内外工具
   * 基于LangGraph构建
   * 支持复杂任务规划执行

   **工具支持**: 可调用Google搜索、文件I/O、Linux Shell等外部功能
